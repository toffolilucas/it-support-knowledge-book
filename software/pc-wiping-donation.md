<h1>Correção do erro: Preparação e formatação segura de hardware legado para doação</h1><br><p>📌 <strong>Descrição do Problema</strong></p><p>Usuário necessita doar/presentear um computador antigo para terceiros, demandando a preservação integral de seus dados pessoais atuais em armazenamento externo (HD Externo/Pen Drive) e a posterior formatação limpa do sistema operacional para entrega segura, livre de dados residuais e com otimização de performance.</p><br><p>💻 <strong>Ambiente</strong></p><p>Dispositivo: Desktop / Laptop Legado</p><p>Contexto: Gerenciamento de Armazenamento / Implantação de Sistema Operacional (OS)</p><br><p>🔍 <strong>Diagnóstico e Tentativas de Solução</strong></p><p>Mapeamento de Dados: Identificação e isolamento do diretório de usuários do sistema operativo para coleta de dados voláteis (Desktop, Documentos, Downloads, Imagens). Validação da integridade do sistema de arquivos do dispositivo de destino (HD Externo).</p><br><p>✅ <strong>Solução Aplicada</strong></p><p>Realizado o backup a frio via cópia direta da árvore de diretórios do usuário para o armazenamento externo. Na sequência, foi executado o processo de 'Sysprep/Reset' nativo do sistema operacional com a diretiva de limpeza lógica completa da unidade de disco (Wipe). O sistema retornou ao estado de OOBE (Out-of-Box Experience), pronto para a configuração inicial pelo novo proprietário.</p>
